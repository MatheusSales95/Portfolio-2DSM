Estrutura do projeto:
- index.html
- css/styles.css
- js/app.js
- assets/profile.jpg

Como usar:
1) Abra index.html no navegador para testar localmente.
2) Troque assets/profile.jpg pela sua foto.
3) Edite INITIAL_DATA em js/app.js para atualizar nome, resumo, projetos e skills.
4) Faça deploy no GitHub Pages (Settings → Pages).
